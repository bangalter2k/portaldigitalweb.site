Política de Privacidade

Introdução

A Política de Privacidade descreve como coletamos, utilizamos e protegemos os dados dos usuários que acessam nossa plataforma.



Coleta de dados

Coletamos apenas informações necessárias para realizar consultas veiculares.
Dados pessoais como nome, e-mail e forma de pagamento podem ser solicitados para concluir transações.
Informações técnicas, como endereço IP e navegador utilizado, podem ser registradas para fins de segurança.



Uso das informações

Os dados coletados são utilizados exclusivamente para gerar relatórios confiáveis.
Informações pessoais não são utilizadas para fins comerciais sem consentimento.
Utilizamos dados agregados para melhorar nossos serviços e identificar padrões de uso.



Compartilhamento

Não compartilhamos informações pessoais com terceiros sem autorização do usuário.
Podemos compartilhar dados com autoridades competentes em caso de exigência legal.
Parceiros tecnológicos podem ter acesso limitado a dados para garantir o funcionamento da plataforma.



Segurança

Utilizamos criptografia e protocolos avançados para proteger informações sensíveis.
Monitoramos constantemente nossos sistemas para prevenir acessos não autorizados.
O usuário também é responsável por manter suas credenciais seguras.



Alterações na política

Podemos atualizar esta política conforme necessário. Alterações significativas serão comunicadas aos usuários.
